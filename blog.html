Question-1: What is DOM & What is the purpose of the DOM?

Ans:The objects that make up a web document's structure and content are represented as data by the Document Object Model (DOM).

The main purpose of DOM is a website document's elements can be added, changed, or removed using this programming interface. DOM manipulation is the process of adding, deleting, and changing webpage elements using JavaScript. In web development, it occurs frequently.


Question-2:How to select an element from DOM?

Ans: (i) getElementById: search element by element_id.

(ii)getElementsByTagName: search element by tag name (e.g., span, div)

(iii)getElementsByClassName: search element by class name.

(iv)getElementsByName: search element by name attribute.


Question-3: What is an event bubble in Javascript?

Ans: The DOM has a concept known as event bubble. The event bubbles up to the element's parent and grandparent elements in the DOM tree until it reaches the root element once an element receives an event.


Question-4:What is a Callback Function & Wat is the use of callback function ?

Ans: A JavaScript callback is a function that is to be run following the completion of another function. A callback function is any function that is supplied as an input to another function so that it can be used to execute that other function, according to a more rigorous definition.

The function of a callback is to run code in response to an event. Using callback, we can add wording at the end of each statement that says "run this code whenever the user touches a key on the keyboard." These events can be like mouse clicks.
